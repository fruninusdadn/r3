# Quick Start

Set up Recall in 2 minutes. This guide covers installation, configuration, and your first memory operation.

## Before you begin

You'll need:
- [Claude Desktop](https://claude.ai/download) installed
- [Node.js 18+](https://nodejs.org) installed
- [Redis](https://redis.io) running locally or in the cloud
- A [Mem0 API key](https://mem0.ai) (free tier available)

## Step 1: Start Redis

```bash
# macOS
brew install redis
brew services start redis

# Docker (all platforms)
docker run -d -p 6379:6379 redis:alpine

# Verify it's running
redis-cli ping
# Should return: PONG
```

## Step 2: Get your Mem0 API key

1. Sign up at [mem0.ai](https://mem0.ai)
2. Go to **Dashboard** â†’ **API Keys**
3. Click **Create new key**
4. Copy the key starting with `mem0_`

## Step 3: Configure Claude Desktop

Add Recall to your Claude Desktop configuration:

```bash
# macOS/Linux
nano ~/.claude/claude_desktop_config.json

# Windows
notepad %APPDATA%\Claude\claude_desktop_config.json
```

Add this configuration:

```json
{
  "mcpServers": {
    "recall": {
      "command": "npx",
      "args": ["@n3wth/recall"],
      "env": {
        "MEM0_API_KEY": "mem0_YOUR_KEY_HERE",
        "REDIS_URL": "redis://localhost:6379"
      }
    }
  }
}
```

## Step 4: Restart Claude Desktop

1. Quit Claude Desktop completely
2. Start Claude Desktop again
3. Open a new conversation

## Step 5: Test it works

In Claude Desktop, type:

```
Remember that my name is Sarah and I prefer Python for data science projects.
```

Claude should respond confirming the memory was stored. Then start a new conversation and ask:

```
What's my name and preferred programming language?
```

Claude should recall your information from the previous conversation.

## What's happening

When you ask Claude to remember something:

1. **Storage**: The memory is stored in Redis (local cache)
2. **Backup**: It's synced to Mem0 cloud (permanent storage)
3. **Retrieval**: Future queries search both cache and cloud
4. **Response**: Relevant memories are returned in &lt;5ms from cache

## Configuration options

### Use Redis Cloud

For production or remote access:

```json
{
  "mcpServers": {
    "recall": {
      "command": "npx",
      "args": ["@n3wth/recall"],
      "env": {
        "MEM0_API_KEY": "mem0_...",
        "REDIS_URL": "redis://default:password@redis-12345.c1.us-east-1.ec2.cloud.redislabs.com:12345"
      }
    }
  }
}
```

### Set a user ID

To separate memories by user:

```json
{
  "env": {
    "MEM0_API_KEY": "mem0_...",
    "REDIS_URL": "redis://localhost:6379",
    "MEM0_USER_ID": "sarah_chen"
  }
}
```

### Enable debug logging

To see what's happening:

```json
{
  "env": {
    "MEM0_API_KEY": "mem0_...",
    "REDIS_URL": "redis://localhost:6379",
    "DEBUG": "recall:*"
  }
}
```

## Common issues

### "Redis connection failed"

Redis isn't running. Start it with:
```bash
redis-server
```

### "Invalid API key"

Check your Mem0 API key:
1. Starts with `mem0_`
2. No extra spaces
3. Copied correctly

### "MCP server not found"

Claude Desktop needs a restart:
1. Quit Claude Desktop (Cmd+Q / Alt+F4)
2. Start it again
3. Check the config file is valid JSON
