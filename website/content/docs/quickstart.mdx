# Quick Start

Get r3call running in 30 seconds - zero configuration required.

## Step 1: Run r3call

```bash
npx r3call
```

That's it! r3call automatically starts with an embedded Redis server. No setup needed.

## Step 2: Configure Claude Desktop

Open your Claude Desktop configuration:

```bash
# macOS/Linux
nano ~/.claude/claude_desktop_config.json

# Windows
notepad %APPDATA%\Claude\claude_desktop_config.json
```

Add this simple configuration:

```json
{
  "mcpServers": {
    "recall": {
      "command": "npx",
      "args": ["r3call"]
      // No environment variables needed!
    }
  }
}
```

## Step 3: Restart Claude Desktop

1. Quit Claude Desktop completely
2. Start Claude Desktop again
3. Open a new conversation

## Step 4: Test it works

In Claude Desktop, type:

```
Remember that my name is Sarah and I prefer Python for data science projects.
```

Claude should respond confirming the memory was stored. Then start a new conversation and ask:

```
What's my name and preferred programming language?
```

Claude should recall your information from the previous conversation.

## What's happening

When you ask Claude to remember something:

1. **Local Storage**: Memory is instantly stored in the embedded Redis server
2. **Persistence**: Data persists between sessions on your local machine
3. **Fast Retrieval**: Memories are retrieved in under 5ms
4. **Zero Dependencies**: Everything runs locally, no external services needed

## Optional: Advanced Configuration

### Enable Cloud Sync

Want to backup memories to the cloud? Add a Mem0 API key:

```json
{
  "mcpServers": {
    "recall": {
      "command": "npx",
      "args": ["r3call"],
      "env": {
        "MEM0_API_KEY": "mem0_..."  // Get free at mem0.ai
      }
    }
  }
}
```

### Use External Redis

For production or shared Redis:

```json
{
  "mcpServers": {
    "recall": {
      "command": "npx",
      "args": ["r3call"],
      "env": {
        "REDIS_URL": "redis://your-redis-server:6379"
      }
    }
  }
}
```

### Set a user ID

To separate memories by user:

```json
{
  "env": {
    "MEM0_API_KEY": "mem0_...",
    "REDIS_URL": "redis://localhost:6379",
    "MEM0_USER_ID": "sarah_chen"
  }
}
```

### Enable debug logging

To see what's happening:

```json
{
  "env": {
    "MEM0_API_KEY": "mem0_...",
    "REDIS_URL": "redis://localhost:6379",
    "DEBUG": "recall:*"
  }
}
```

## Common issues

### "Redis connection failed"

Redis isn't running. Start it with:
```bash
redis-server
```

### "Invalid API key"

Check your Mem0 API key:
1. Starts with `mem0_`
2. No extra spaces
3. Copied correctly

### "MCP server not found"

Claude Desktop needs a restart:
1. Quit Claude Desktop (Cmd+Q / Alt+F4)
2. Start it again
3. Check the config file is valid JSON
